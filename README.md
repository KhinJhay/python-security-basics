# python-security-basics# Python Security Basics: Environment Variables

This repository contains the companion code for the tutorial on securing Python applications using `python-dotenv`.

## How to use this code
1. Clone the repository.
2. Install the requirements: `pip install python-dotenv requests`
3. Rename the `.env.example` file to `.env`
4. Add your own dummy API keys to the `.env` file.
5. Run the script: `python secure_practice.py`

*Note: Never commit your actual `.env` file to version control.*
